### Overview of steps required

There are only three simple steps to get started.

---

<div class="gs-section-01" markdown="1">

### 1. Fork the Beautiful Jekyll repository

Fork the repository by clicking the Fork button on GitHub.

</div>

---

<div class="gs-section-02" markdown="1">

### 2. Build your website

Edit the _config.yml file and customize your site.

</div>

---

<div class="gs-section-03" markdown="1">

### 3. Deploy with GitHub Pages

Activate GitHub Pages in the repository settings.

</div>
